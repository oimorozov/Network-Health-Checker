# Network Health Checker

Простая CLI-утилита для пинга на L4 TCP уровне серверов

Использование:
```console
$ go run main.go <file>
```

Пример входного файла:
```txt
Имя1,IP1
Имя2,IP2
...
ИмяN,IPN
```
